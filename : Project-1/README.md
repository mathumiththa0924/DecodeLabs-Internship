# ☕ Project 1 — Responsive Frontend Interface

**DecodeLabs Internship 2026 | Full Stack Development Track**

---

## 📌 Project Overview

A fully responsive landing page for **Brewed & Co.**, an artisan café. Built using pure HTML5, CSS3, and vanilla JavaScript — no frameworks used. This project demonstrates responsive design principles, semantic markup, and clean UI implementation.

---

## 🎯 Goal

Create a responsive frontend interface for a simple web application that works seamlessly across all device sizes.

---

## ✅ Key Requirements Met

- ✅ Built with **HTML5, CSS3, and JavaScript** only (no frameworks)
- ✅ **Responsive layout** for mobile, tablet, and desktop
- ✅ **Clean and user-friendly UI**
- ✅ Semantic HTML5 landmarks (`<nav>`, `<section>`, `<footer>`)
- ✅ Mobile hamburger menu with JavaScript toggle
- ✅ CSS Grid and Flexbox for layout
- ✅ Smooth scroll and active nav highlighting

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure and semantic markup |
| CSS3 | Styling, Grid, Flexbox, animations |
| JavaScript | Mobile menu, scroll effects, interactivity |
| Google Fonts | Playfair Display + Inter typography |

## 📱 Responsive Breakpoints

| Screen Size | Layout |
|-------------|--------|
| Mobile (< 768px) | Single column, hamburger menu |
| Tablet (768px–1024px) | 2-column grid |
| Desktop (> 1024px) | Full multi-column layout |
